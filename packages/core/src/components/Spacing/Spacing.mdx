# Spacing

The Spacing component provides a visual representation of spacing tokens in the design system, helping maintain consistent spacing throughout the application.

## Features

- Visual representation of spacing tokens
- Multiple size options (xs to 4xl)
- Vertical and horizontal directions
- Customizable colors
- Pixel value display
- Interactive controls

## Usage

### Basic Usage

<Source
  language="tsx"
  code={`
import { Spacing } from '@crossuikit/core';

function MyComponent() {
  return (
    <div>
      <h1>Title</h1>

  `}

## Best Practices

1. **Consistency**: Use consistent spacing sizes throughout your application.
2. **Visual Hierarchy**: Use larger spacing for major sections and smaller spacing for related elements.
3. **Responsive Design**: Consider how spacing affects layout on different screen sizes.
4. **Accessibility**: Ensure spacing doesn't create usability issues for users with disabilities.
5. **Documentation**: Use the showValue prop during development to understand spacing values.
6. **Color Usage**: Use subtle colors for spacing indicators to avoid visual noise.
7. **Direction**: Choose the appropriate direction (vertical/horizontal) based on your layout needs.

## Spacing Scale

The design system uses a consistent spacing scale:

| Size | Value |
|------|-------|
| xs   | 4px   |
| sm   | 8px   |
| md   | 16px  |
| lg   | 24px  |
| xl   | 32px  |
| 2xl  | 48px  |
| 3xl  | 64px  |
| 4xl  | 96px  |
