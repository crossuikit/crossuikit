## Getting Started

### Installation

```bash
npm install @crossuikit/core
# or
yarn add @crossuikit/core
# or
pnpm add @crossuikit/core
```

### Basic Usage

```tsx
import { Button } from '@crossuikit/core';

function App() {
  return (
    <Button variant="primary">
      Click me
    </Button>
  );
}
```

### Theme Provider

To enable dark mode and theming, wrap your app with the `ThemeProvider`:

```tsx
import { ThemeProvider } from '@crossuikit/core';

function App() {
  return (
    <ThemeProvider>
      {/* Your app content */}
    </ThemeProvider>
  );
}
```
